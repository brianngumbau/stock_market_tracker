About Stock Market Tracker

Stock Market Tracker is a web application designed to help users track stock market data efficiently. Users can:

  • Search for stocks and view detailed data such as open, close, high, low, and volume.
  • Add stocks to their watchlist to monitor their performance over time.
  • View stock charts for visual insights.

This project utilizes the following technologies:

  • Backend: Flask (Python)
  • Frontend: Bootstrap
  • Database: MongoDB Atlas
  • Stock Data: yfinance API

Future enhancements include real-time stock data, market trends, and notifications for stock price changes.

Developed by Brian Ngumbau as part of the Webstack Portfolio Project.
